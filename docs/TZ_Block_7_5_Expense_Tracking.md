# Блок 7.5: Учёт фактических расходов (Expense Tracking)

## 1. Общие принципы

- **Доступ:** Через отдельную команду после ввода пароля
- **Роли:** Менеджер, Админ, CEO, Суперюзер (Клиент и Агент — исключены)
- **Валюта:** Только рубли (₽)
- **Ввод:** Свободный, без лимитов и блокировок
- **Уведомления:** Отключены (кроме напоминания @vivacarmen на следующий день после мероприятия)
- **Редактирование:** Всегда, с полным логом изменений. Только CEO может редактировать чужие записи

## 2. Типы расходов

### Категории (все используются)

| Код | Название | Описание |
|-----|----------|----------|
| CONTRACTOR | Подрядчики | Звук, свет, декор, кейтеринг, фото/видео и т.д. |
| TRANSPORT | Транспорт | Доставка, такси, аренда авто, бензин |
| TEAM_FOOD | Питание команды | Обеды, кофе, вода, перекусы на площадке |
| MATERIALS | Материалы | Расходники, ремонт, мелкая фурнитура |
| EMERGENCY | Непредвиденные | Форс-мажор, замена оборудования, доп. услуги |
| OVERHEAD | Общие/Накладные | Аренда офиса, зарплата, коммунальные — не привязаны к проектам |

## 3. Уровни привязки

### Уровень 1: Привязанные к смете
- Расход относится к категории в смете (например, "звук")
- Не привязан к конкретной строке — свободное соответствие
- Пример: Оплатили звукорежиссёра → категория "Подрядчики: звук"

### Уровень 2: Общие (не привязанные)
- Расход без привязки к проекту
- Используется только для категории OVERHEAD
- В отчётах отдельная статья "Компания"

## 4. Способы оплаты

| Код | Название | Примечание |
|-----|----------|------------|
| CASH | Наличные | |
| CARD | Карта | Личная или корпоративная |
| ACCOUNT | С расчётного счёта | Перевод юрлицу |
| OTHER | Свой вариант | Ручной ввод |

## 5. Процесс внесения расходов

### Шаг 0: Доступ
```

```

### Шаг 1: Команда
```
/expense
```
Или кнопка: [💰 Внести расход]

### Шаг 2: Выбор режима ввода

```

```

### Вариант А: Пошаговый ввод

```

```

## 7. Редактирование расходов

### Свои расходы
```
Мои расходы:
[EXP-0340] 20.03.25 — 45 000 ₽ — Подрядчики [✏️ Изменить] [🗑 Удалить]
[EXP-0341] 22.03.25 — 3 500 ₽ — Транспорт [✏️ Изменить] [🗑 Удалить]
```

### Чужие расходы (только CEO)
```
/expense_edit EXP-0340

Текущие данные:
Сумма: 45 000 ₽
Изменить на: ________

[💾 Сохранить]

⚠️ Изменение зафиксировано в логе. Уведомление отправлено автору (Петрова).
```

## 11. Интеграции

### С Блоком 7 (Аналитика)
- Расходы включаются в финансовые отчёты
- Метрика "Реальная прибыль" = Выручка − Фактическая себестоимость

### С Блоком 5 (Документооборот)
- Внутренний акт для CEO: приложение "Фактические затраты"

### С Блоком 8 (Интеграции внешние)
- Выгрузка в 1С: расходы по проектам + общие расходы отдельно

## 12. Что НЕ реализуется

| Функция | Причина |
|---------|---------|
| Загрузка фото чеков | Только сумма и комментарий |
| Автопроверки лимитов | Свободный ввод |
| Уведомления о превышении | Только напоминание @vivacarmen |
| Связь с задачами в плане проекта | Отдельный процесс |
| Распределение общих расходов на проекты | Отдельная статья "Компания" |
| Мультивалютность | Только ₽ |
