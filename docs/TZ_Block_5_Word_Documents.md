# Блок 5: Документооборот (Word)

## 1. Типы документов

| Документ | Формат | Описание |
|----------|--------|----------|
| Договор оказания услуг | .docx | Основной документ |
| Приложение (спецификация) | .docx | Копия сметы в таблице |
| Акт выполненных работ | .docx | Черновик (создаётся сразу) |
| Счёт на оплату | .docx | Для бухгалтерии клиента |

## 2. Шаблоны договоров

| Тип клиента | Шаблон | Выбор |
|-------------|--------|-------|
| Прямой клиент | Универсальный (все бренды) | Автоматически |
| Агент | Отдельный (другие условия) | Автоматически по роли |
| IM-клиент | Шаблон IM (отдельный) | Автоматически по бренду IM |

Шаблоны предоставляет CEO: Word-файлы с метками `{{поле}}` для замены.

## 3. Номер договора

**Формула:** `I[порядковый номер]-[мм][гг]`

**Примеры:**
- Первый договор в марте 2025: `I1-0325`
- Десятый договор в марте 2025: `I10-0325`

**Отслеживание:** Счётчик в БД, сброс при смене месяца.

## 4. Источники данных для заполнения

**Из сметы (финальной):**
- Номер договора (генерируется)
- Дата договора
- Название проекта
- Сумма (цифрами и прописью)
- Состав услуг (таблица)

**Из карточки компании (парсинг Word/PDF):**
- Полное наименование заказчика
- ИНН, КПП, ОГРН
- Юридический адрес
- Банковские реквизиты (БИК, счёт, наименование банка)
- ФИО и должность подписанта
- Основание подписания (Устав/Доверенность)
- Email (опционально)
- Телефон (опционально)

**Запрашиваем отдельно (если нет в карточке):**
- ФИО подписанта
- Должность подписанта
- Телефон для связи

## 6. Приложение (спецификация)

**Формат:** Таблица Word с копией финальной сметы

**Структура:**
- Таблица как в Excel (без колонок себестоимости)
- Итоговые строки: Сумма услуг, Агентское 10%, Налоги, Всего
- Подпись: "Составлено менеджером", "Согласовано заказчиком"

## 8. Формирование счёта (детально)

### Вариант "Кратко (по умолчанию)"

**Формулировка:**
```
Оплата за мероприятие "[Название проекта]" 
согласно Приложению №[номер] от [дата] 
к Договору №[номер] от [дата] 
[Предлог] [Место проведения] [Дата проведения]
```

### Правила предлогов и сокращений

| Место проведения (как в смете) | В счёте | Предлог |
|--------------------------------|---------|---------|
| Курорт Завидово | Завидово | в |
| Завидово | Завидово | в |
| Москва | Москва | в |
| Санкт-Петербург | Санкт-Петербург | в |
| Другое | Как есть | на территории / в |

**Примеры:**
- `... в Завидово 25.03.2025`
- `... в Москве 25.03.2025`
- `... на территории загородного клуба "Лесная поляна" 25.03.2025`

### Источники данных для счёта

| Поле | Откуда |
|------|--------|
| Название проекта | Поле "Название проекта" из шапки сметы (как есть) |
| Приложение № | Номер приложения (1, 2, 3...) |
| Дата приложения | Дата договора |
| Договор № | Сгенерированный I[N]-[мм][гг] |
| Дата договора | Текущая дата |
| Место проведения | Поле "Место проведения" из шапки (с сокращением) |
| Дата проведения | Поле "Дата и время" из шапки |
